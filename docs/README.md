# Documentación - QUADRA

Esta carpeta contiene la documentación técnica y de features del proyecto QUADRA.

## 📚 **Documentos disponibles:**

### 🗺️ **Features del mapa:**
- **`GEOLOCATION_FEATURES.md`** - Documentación de las funcionalidades de geolocalización
- **`DASHBOARD_MAP_FEATURES.md`** - Documentación de las features del mapa en el dashboard

## 📋 **Estructura del proyecto:**

```
docs/
├── README.md                    # Este archivo
├── GEOLOCATION_FEATURES.md     # Features de geolocalización
└── DASHBOARD_MAP_FEATURES.md   # Features del dashboard
```

## 🔍 **Contenido:**

### Geolocalización:
- Implementación de detección automática de ubicación
- Manejo de errores de geolocalización
- Integración con Leaflet.js
- Configuración de marcadores personalizados

### Dashboard:
- Funcionalidades del mini-mapa (removido)
- Integración con Bootstrap modals
- Sistema de navegación entre vistas
- Configuración de tiles y mapas

## 📝 **Para desarrolladores:**
Esta documentación describe las decisiones técnicas, implementaciones específicas y lecciones aprendidas durante el desarrollo de las funcionalidades de mapas y geolocalización en QUADRA.
